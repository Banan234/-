Общие модули и статичные справочники, которые используются несколькими слоями:
frontend (`src/`), backend (`server.js`, `lib/`) и import scripts (`scripts/`).

Правило: сюда кладём только код и данные без зависимости от браузера, Express,
Vite или файловой системы. Runtime/generated данные остаются в `data/`.
