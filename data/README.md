Папка для импортируемых данных.

Статичные общие справочники, которые нужны и фронту, и серверу, лежат в
`../shared/` (например, `catalogCategories.json`). Здесь остаются входные и
сгенерированные данные импорта.

Файлы:

- `price.xlsx` или `price.xls` — исходный прайс поставщика
- `products.json` — результат работы `scripts/importPrice.js`
- `importConfig.json` — настройки импорта: fallback-пороги подозрительных цен,
  параметры расчёта p95/p99 по истории и лимиты диагностик классификации.
- `import-history.json` — сгенерированная история последних импортов для
  адаптивного расчёта порогов подозрительных цен.
- `import-report.json` — отчёт о последнем импорте: сводка, распределение по
  категориям, пропущенные строки с причинами и diff относительно предыдущего
  импорта (что добавилось, удалилось, изменились цены/остатки). Генерируется
  автоматически при каждом запуске `npm run import:price`.
- `import-report.html` — человекочитаемая версия отчёта с вкладками для
  добавленных, удалённых, резких изменений цен, подозрительных цен и смен
  категорий. Генерируется автоматически вместе с JSON-отчётом.
